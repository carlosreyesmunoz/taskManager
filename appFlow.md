# Task Manage Flow

This app help groups or families to organise the work of home or project.

## FLOW

An admin user can create tasks, add task to a task pool or assign tasks to users including itself.
Users can get tasks from the task pool.
Once a task is completed the user mark the task as completed,
then the Admin that create the task can verify completion and mark task as finalised.
Task can have points that are added to the user profile.

Users must be part of an organization, users of one organization can not see tasks or users from other organisation.
Admin can invite users to the organisation and assign user role.
